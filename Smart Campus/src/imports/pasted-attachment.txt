Create a fully functional Smart Campus Networking Platform web application designed for students within a college campus to connect, collaborate, and share opportunities. The website must be fully interactive with all buttons and actions working correctly. Users must be able to create accounts, update information, send messages, and interact with the platform without losing data after refresh.

General Requirements

The website must be fully functional and interactive, not just a UI prototype.

All buttons, forms, and navigation elements must work properly.

Users must be able to create usernames, edit profiles, and update their data.

Any changes made by users must update instantly and persist after page refresh.

The system must support custom Dark Mode and Light Mode themes with a toggle switch.

The interface must be modern, clean, and student-friendly.

Use smooth transitions, hover effects, and responsive design.

Design Requirements

Clean modern UI with glassmorphism style translucent navigation bar

Rounded cards, subtle shadows, and minimalistic layout

Smooth animations and interactive elements

Fully responsive design for desktop, tablet, and mobile

Theme toggle allowing users to switch between custom dark and light themes

Core Features and Pages
Authentication System

Create a secure user authentication system.

Functions:

User registration (create username, email, password)

Login system

Logout functionality

User session management

User accounts must be stored in a database.

Dashboard

After login, users should see a personal dashboard displaying:

Welcome message with user name

Campus announcements

Recently active students

Upcoming events

Quick navigation cards

Student Networking Page

Students should be able to:

View profiles of other students

Connect with other students

View shared interests or skills

Send connection requests

Messaging System

Users should be able to:

Send messages to other students

View chat conversations

Receive notifications when new messages arrive

Skill Sharing Section

Students can:

List their skills

Offer help or collaboration

Find other students with similar skills

Opportunities Board

Allow students to post and view:

Internship opportunities

Project collaboration

Events

Workshops

Users should be able to add, edit, or delete their posts.

Profile Management

Users must be able to:

Change username

Update profile information

Add skills and interests

Upload profile picture

Save changes permanently

Notifications System

Users receive notifications for:

Messages

Connection requests

Event updates

Collaboration requests

Frontend Technologies

Use modern frontend technologies:

React.js

Vite

Tailwind CSS

React Router

Use reusable components such as:

Navbar

Profile cards

Message components

Event cards

Notification panels

Backend Requirements

Implement a backend to manage users and interactions.

Recommended technologies:

Node.js

Express.js

MongoDB database

Backend should manage:

User accounts

Messages

Connections

Events

Skill listings

Notifications

Each user must only access their own account data.

Data Persistence

All user activities must be saved in the database so that:

User profiles remain saved

Messages remain saved

Posts remain saved

Changes remain after page refresh

Performance Requirements

Fast loading pages

Smooth UI interactions

Efficient API communication

Optimized database queries

Goal

Create a complete campus networking platform similar to a private LinkedIn for college students, enabling collaboration, communication, and opportunity sharing within the campus community.

The system must look professional, modern, and suitable for a final year project demonstration.
